## 项目简介

本项目为 **2023年亚太杯数学建模竞赛（APMCM）** 参赛论文及相关代码，选题为 **C题：基于回归分析与预测模型的新能源汽车分析与未来发展趋势研究**。

论文围绕新能源汽车在中国及全球的发展现状、影响因素、未来预测、政策影响及环境效益等方面展开建模与分析，提供了完整的数据处理、模型建立、结果可视化与代码实现。


##  主要内容概述

### 问题一：影响中国新能源汽车发展的主要因素分析
- **使用模型**：多元线性回归模型
- **自变量**：充电桩数量、动力电池装机量、产业链投资额、碳排放、人均国民总收入
- **因变量**：新能源汽车销量
- **主要结论**：新能源汽车发展主要受充电桩基础设施和动力电池装机量影响
- **代码文件**：`第一问题代码.py`

### 问题二：中国新能源汽车未来十年发展趋势预测
- **使用模型**：ARIMA时间序列预测模型
- **关键技术**：数据平稳性检验、差分处理、模型定阶（AIC准则）
- **代码文件**：`第二问题代码.py`

### 问题三：新能源汽车对全球传统能源汽车产业的影响分析
- **分析方法**：Pearson相关系数检验 + Granger因果检验 + 向量自回归模型（VAR）
- **分析因素**：全球油价、传统能源技术研发、新能源汽车销量
- **代码文件**：`第三问题代码.py`

### 问题四：国外针对性政策对中国新能源汽车发展的影响分析
- **使用模型**：灰色预测模型（GM(1,1)）
- **方法**：对比政策发布前后的实际值与预测值
- **结论**：国外政策对中国新能源汽车发展影响有限
- **代码文件**：`第四问题代码.py`

### 问题五：城市新能源汽车电动化对生态环境的影响分析
- **分析对象**：百万人口城市
- **计算方法**：基于车辆保有量推算碳排放
- **重要结果**：电动化后碳排放整体下降约28%

### 问题六：致市民的公开信
- **目的**：基于分析结论宣传新能源汽车的环保效益
- **内容**：强调电动化对空气质量和能源结构改善的贡献
