项目简介
本项目为2023年亚太杯数学建模竞赛（APMCM）C题的完整解决方案。围绕新能源汽车的发展趋势，通过建立多种数学模型，系统分析了影响因素、未来预测、产业影响、政策效应及环境效益，并撰写公开信进行科普宣传。

研究内容概述
问题	方法	主要结论
1. 影响因素分析	多元线性回归	充电桩数量、电池装机量影响显著
2. 未来十年预测	ARIMA时间序列	销量将持续增长，2025年预计突破XX万辆
3. 对传统汽车产业影响	Pearson + VAR	存在替代效应，但传统技术研发仍保持正相关
4. 国外政策影响评估	灰色预测 + 对比分析	政策影响有限，中国新能源汽车出口仍保持增长
5. 城市电动化环境效益	生命周期碳排放计算	电动化后碳排放降低约28%
6. 公开信撰写	结论归纳 + 科普表达	强调新能源汽车的环保与能源转型价值
