亚太杯数学建模竞赛论文与代码 - 新能源汽车发展分析
项目简介
本项目为 2023年亚太杯数学建模竞赛（APMCM） 参赛论文及相关代码，选题为 C题：基于回归分析与预测模型的新能源汽车分析与未来发展趋势研究。论文围绕新能源汽车在中国及全球的发展现状、影响因素、未来预测、政策影响及环境效益等方面展开建模与分析，提供了完整的数据处理、模型建立、结果可视化与代码实现。

团队信息
选题：C题
研究方向：新能源汽车发展建模与预测
奖项：三等奖（上面有官方证书）

主要内容
问题一：影响中国新能源汽车发展的主要因素分析
使用多元线性回归模型

自变量：充电桩数量、动力电池装机量、产业链投资额、碳排放、人均国民总收入

因变量：新能源汽车销量

代码文件：第一问题代码.py

问题二：中国新能源汽车未来十年发展趋势预测
使用ARIMA时间序列预测模型

数据预处理、平稳性检验、差分处理、模型定阶与诊断

代码文件：第二问题代码.py

问题三：新能源汽车对全球传统能源汽车产业的影响分析
Pearson相关系数检验 + Granger因果检验 + 向量自回归模型（VAR）

分析因素：全球油价、传统能源技术研发、新能源汽车销量

代码文件：第三问题代码.py

问题四：国外针对性政策对中国新能源汽车发展的影响分析
采用灰色预测模型（GM(1,1)）

对比政策发布前后的实际值与预测值

代码文件：第四问题代码.py

问题五：城市新能源汽车电动化对生态环境的影响分析
基于百万人口城市进行车辆保有量推算

计算电动化前后碳排放与能耗变化

结果显示电动化后碳排放整体下降约28%

问题六：致市民的公开信
基于上述分析结论撰写

宣传新能源汽车的环保效益与社会贡献
